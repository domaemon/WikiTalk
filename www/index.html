Language Preferencee

Set the language of your prefrence. It will be used to specify Wikipedia locale and also the speech language.

Continuous Mode

When on, WikiTalk continuously jumps to and reads the articles. When off, WikiTalk reads only one article.

Random Mode

When on, WikiTalk randomly selects the content for you. When off, WikiTalk continues to choose related topics. The initial topic will be chosen randomly regardless of the settings.

Voice Speed

You can set how fast WikiTalk reads. The default speed is 1.0x.
