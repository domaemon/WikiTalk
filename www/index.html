Summary Mode

When on, WikiTalks reads only the summary of the article. When off, WikiTalk reads all the paragraphs within the same article.

Continuous Mode

When on, WikiTalk continuously jumps to and reads the related articles. When off, WikiTalk reads only one article.

Random Mode

When on, WikiTalk randomly select the initial content for you. When off, WikiTalk asks you to voice the title of the article.

Voice Speed

You can set how fast WikiTalk reads. The default speed is Normal.
